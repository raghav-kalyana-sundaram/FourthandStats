# Simulations (Phase 6 — Future)

The simulations module will implement:

- **Matchup Simulator** — Monte Carlo game simulation using team EPA ratings and variance
- **Season Simulator** — projected standings and playoff seed probabilities
- **Playoff Bracket Simulator** — full playoff bracket odds
- **What-If Team Simulator** — swap roster components and re-simulate

These are not yet implemented. The module stubs are in place so the package structure is stable.

See `docs/simulation_methodology.md` for the planned methodology.
